Discuss the scope of `var`, `let`, and `const`.

VAR:

  • Use it when we want a variable to be accessible throughout an entire function or the whole program.
  • Can be changed later if needed.

LET:

  • Use it when we want a variable to be limited to a specific block of code.
  • Can be changed later if needed.

CONST:

  • Use it when we want a variable to stay the same (not change) within a block of code.
  • For numbers and text, it can't be changed. For lists of things, we can add or remove items, but we can't replace the whole list.

Tell us the use cases of `null` and `undefined`

NULL:

  • Think of it as an intentional empty value. We use it when we want to say, This thing has no value on purpose.

UNDEFINED:

  • Think of it as the default empty value. It's what we get when something hasn't been given value. If we see undefined, it often means no one set a value here, or something doesn't exist.
